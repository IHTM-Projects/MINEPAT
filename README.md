# MINEPAT
Learning Projects to learn and practice on a real world scenario, where good practices patterns are met and applied to build scalable platforms.

## Repository content
It's splitted into two main parts or apps:
- The back office
- The front office

Each owns a dedicated branch in this repo.

## Project Tools

#### Front office 
It's a mobile app built with the following:
- Flutter

#### Back office
The backend and the functionalities of the app, built with the following:
- Fast API
- Python
- Postgres database powered by Neon DB (cloud database manager)
- Pydantic
